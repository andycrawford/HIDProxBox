Feature Highlights

Everything in the daemon, no third-party services required.

4-Port Instant Switching

Dedicated selector buttons with LED confirmation. Every computer stays USB-connected — zero warm-up delay on switch.

INPUT

Wired + Wireless Input

Accept keystrokes from a USB keyboard via evdev, or any Bluetooth HID keyboard. Toggle the active source with one button press.

INPUT

Programmable Macros

Map HID keystroke sequences — lock screen, screenshot, custom combos — to button short- or long-press actions via config.py.

INPUT

Zero Target Software

Each CH552T enumerates as a standard USB HID keyboard and mouse. Your computers never know it is a proxy.

INPUT

USB + Bluetooth Output

Relay via CH552T HID devices over USB, or turn the Pi into a Bluetooth keyboard for cable-free control. Per-computer mode persists across reboots.

OUTPUT

Browser Control Panel

Switch computers from any phone, tablet, or browser at http://<pi-ip>:8080. No app required — just your local network.

OUTPUT

Paste Text Entry

Type or paste any text into the browser panel and send it as keystrokes to the active computer — ideal for passwords at BIOS prompts, full-disk encryption screens, or VMs with isolated clipboards.

OUTPUT

Optional Touchscreen

Local tkinter GUI with tally lights and one-tap switching. Includes a built-in ZSA Keymapp launcher for ZSA keyboard users.

OUTPUT

Open-Source & Hackable

Producer-consumer architecture — add a new input source or output sink in a few dozen lines of Python. MIT licensed.

OUTPUT

Security

The Password Problem

BIOS screens, full-disk encryption prompts, and VMs with isolated clipboards all block clipboard paste. The web panel's Paste Text feature types your password character by character over HID — no clipboard access needed on the target.

Paste in browser → keystrokes arrive at the target instantly

Get Started

Clone, install, run. Flash the firmware. Wire and configure.

1

Clone and install

git clone https://github.com/andycrawford/HIDProxBox.git /opt/hidproxbox
pip3 install evdev pyserial pybluez --break-system-packages
sudo apt-get install bluetooth bluez python3-dbus python3-gi
2

Flash the firmware

Flash firmware/hidproxbox.ino to each CH552T using the Arduino IDE with CH55x board support. One CH552T per computer.

3

Wire and configure

Connect CH552T units via USB-serial adapters. Wire tactile buttons and LEDs to GPIO. Update serial ports, GPIO pins, and macros in config.py.

4

Run the daemon

sudo python3 /opt/hidproxbox/daemon.py --foreground

# Or install as a systemd service
sudo systemctl start hidproxbox
sudo systemctl enable hidproxbox
